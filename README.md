# Portafolio MR Soft

## Descripción
**Portafolio MR Soft** es una aplicación web personal desarrollada con el objetivo de presentar de manera profesional mi perfil, hoja de vida, conocimientos técnicos y experiencia adquirida durante mi formación en **Ingeniería de Software con Inteligencia Artificial**.

El portafolio está diseñado para ser visualmente atractivo, fácil de navegar y adaptable a distintos dispositivos, funcionando como carta de presentación profesional.

## Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript
- Git / GitHub

## Funcionalidades
- Presentación de perfil personal y profesional
- Sección de habilidades y conocimientos técnicos
- Navegación mediante enlaces internos
- Diseño responsive adaptable a móviles y escritorio
- Formulario de contacto
- Uso iconos optimizados

## Estructura del Proyecto
- `css/` → estilos del sitio web
- `cv/` → archivos relacionados a la hoja de vida
- `img/` → imágenes y recursos gráficos
- `main/` → scripts JavaScript y lógica de interacción
- `index.html` → archivo principal del portafolio

## Cómo Ejecutar el Proyecto
1. Clonar el repositorio: git clone https://github.com/MReyes-Soft/portafolio-mr-soft.git
2. Abrir el proyecto en un editor de código.
3. Ejecutar el archivo `index.html` en un navegador web.
4. Opcionalmente, usar un servidor local (Live Server, XAMPP, etc.) para una mejor experiencia.
